# 一、文档概述

## 1.1 编写目的

## 1.2 项目背景

随着人工智能技术的迅猛发展，AI行业资讯更新频繁。短视频平台如抖音和快手已成为信息传播的重要渠道。本项目旨在构建一个智能体系统，实现AI新闻的自动采集、内容生成、视频制作及自动发布。

* 自动抓取最新AI行业新闻。
* 基于大模型生成短视频内容。

* 自动发布至抖音和快手平台。
* 构建可扩展的智能体架构。

### 职责

### 核心功能

* 新闻管理接口

## 3.3 Agent编排层（Agent Orchestration Layer）

**技术栈：AgentScope**

### Agent设计

| Agent名称            | 职责     |
| ------------------ | ------ |
| NewsCollectorAgent | 抓取AI新闻 |
| NewsFilterAgent    | 筛选热点内容 |
| ScriptWriterAgent  | 生成视频脚本 |
| SubtitleAgent      | 生成字幕   |
| VideoPlannerAgent  | 视频生成规划 |
| PublisherAgent     | 发布策略决策 |
| ComplianceAgent    | 内容合规审核 |

---

## 3.4 任务执行层（Task Execution Layer）

### 核心任务

* 新闻采集
* AI内容生成

* 自动发布
